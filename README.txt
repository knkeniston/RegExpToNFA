This program creates an NFA given one or more regular expressions. It 
printes the created NFA out as a table.

To build the program, cd into the folder and run the following commands:
g++ project1.cpp -Wall -ansi -o project1
./project1

List of files in the archive:
- project1.cpp: The project file which creates and prints the NFA.
- test.txt: The file where the regular expressions are written.
- README: The README file with instructions.
